We use really high-level abstractions

For Example

  • Caching provided by
    • web framework
    • cache abstraction library
    • cache engine language bindings
  • Chat clients are GUIs
  • Browsers hide the HTML let alone the HTTP

Many of these have

  • Human-readable(-ish)
  • Simple transmission mechanisms (TCP)

Transmission Control Protocol (TCP)

  • Backbone of web communication
  • Ordered
  • Reliable
  • Error checked
  • Bi-directional

TCP Clients

telnet

  • venerable client
  • nuisance
  • not always installed anymore
$ telnet localhost 6379

netcat (nc)

  • simple
  • attaches stdin/stdout
  • useful for pipes
$ echo "..." | nc localhost 11211 | jq .

socat

  • like netcat but more features
  • especially useful for ssl
$ socat - OPENSSL:duckduckgo.com:443 | jq .
$ socat
